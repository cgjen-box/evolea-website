| T-01-07 | Denial of availability | Font 404 on a build base path | mitigate | @font-face and preload urls are base-prefixed via import.meta.env.BASE_URL, so they resolve on both Cloudflare (/) and GitHub Pages (/evolea-website); dist greps assert correct urls on each build. |
| T-01-SC | Tampering | npm/pip/cargo installs | mitigate | No package installs in this plan. The only external fetch is the OFL Poppins-Bold woff2 from fonts.gstatic.com (validated by woff2 magic bytes); no package-manager legitimacy checkpoint applies. |
